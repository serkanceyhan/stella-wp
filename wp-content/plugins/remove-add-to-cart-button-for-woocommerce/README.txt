Remove Add to Cart Button for WooCommerce plugin gives you a really easy interface to hide/remove the product Add to Cart button and product price.

== Description ==

If you want to hide/remove the product Add to Cart button from the product list pages, product details page and WooCommerce blocks product grid item HTML using any WordPress plugin, please first think about this plugin. Here you have options to remove the productâ€™s Add to Cart button for all users or only for the visitors. If you select the Remove Add to Cart button only for the visitor, then you will see a checkbox where you can decide to show the login button when the Add to Cart button is hidden. Also, have the option to add a custom message, which will be shown when the Add to Cart button is hidden.

In this plugin, there is a settings section from which you can add your login page URL (if you do not enter any URL here, then it will take you to the default login/register page URL) and change the login button label. In advance, have an option to hide the product price. Also, it has options to prevent access to the cart and checkout page and set a redirect URL when trying to access those pages. In addition, the plugin has options to change the default Out of Stock and Backorder messages for products.

== Installation ==

1. Unzip the plugin and upload the `remove-add-to-cart-button-woocommerce` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To change/modify plugin global settings, go to `WooCommerce > Settings > Visibility Settings` tab from Wordpress admin area
4. You are ready to use plugin

== Frequently Asked Questions ==

= Does it remove the Add to Cart button only for the visitors? =

Yes, it has an option to remove the  Add to Cart button only for visitors. Also, have the option to remove the Add to Cart button for all.

= Does it show the login button when the Add to Cart button is removed for the visitors? =

Yes, the login button will be shown instead of the Add to Cart button if you checked this option from the admin area. Also, you can add your message when the Add to Cart button is hidden.
